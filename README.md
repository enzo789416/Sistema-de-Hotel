# Sistema-de-Hotel feito somente em PHP e Mysql
### Caracteristicas do sistema: 
- PHP 7.0
- Mysql 8.0
- SGBD PHPMyadmin
- Laravel 5.8

### Funcionalidades
- The registered user can register, update and delete
